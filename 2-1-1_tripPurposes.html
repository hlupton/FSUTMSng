Trip Purposes

In February 2020, the Model Task Force adopted the trip purposes prepared for FDOT as part of the process of developing standards for FSUTMS1. This document provides some needed specificity on guidelines for trip purposes and includes the following recommendations when developing trip purposes:

  • The models need to produce the information needed for a variety of types of planning analyses, and the models need to be sensitive to a number of factors to perform these analyses.
  • Changes from current practice should be as little as possible and related only to what is necessary to perform the required analyses.
  • It is important to recognize that travel behavior differs by trip purpose, and differs substantially among some types of trip purposes.
  • Increasing knowledge sharing making it easier to apply lessons learned from one model to another, and
  • Trip purpose segmentation must be supported by the data available for model development and validation.

Guidance for Model Developers

  • Make sure that standard trip purposes are used in the model.
  • If deviations from the standard trip purposes are required, document why additional or fewer trip purposes are being proposed.
  • Ensure that there are sufficient observations in the household travel survey data to ensure that all the proposed purposes can be estimated, calibrated, and validated reasonably.
  • If the model accounts for other types of trips (freight, visitors, external, special generators, etc.) besides internal passenger trips, make sure that there is sufficient data to meet those needs.
  • Good practice to develop summary statistics from the survey data (both sample and population) by trip purpose to identify issues with sample being used for model estimation.

Guidance for Model Appliers

  • Make sure that the model has the standard trip purposes.
  • Since access to the underlying survey data might not be available, refer to the validation documentation to ensure that person trip tables by purpose are consistent with underlying data used for model estimation purposes.
  • Be aware of model limitations when the standard trip purposes are not used since any changes to the trip purpose definitions would require re-estimating and recalibrating/validating the model.

Guidance for Model Reviewers

  • Make sure that the proposed trip purposes are consistent with the standard trip purposes; if not, ensure that the reasons for having different trip purposes are justified.
